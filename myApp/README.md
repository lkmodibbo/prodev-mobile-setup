# Mobile Development Environment Setup

## Tools Installed
- Node.js LTS
- Visual Studio Code
- Operating System: Windows
- Expo Go (Android)

## Setup Process
1. Installed Expo Go from Google Play Store.
2. Created an Expo project using `npx create-expo-app`.
3. Started the development server with `npx expo start`.
4. Scanned the QR code using Expo Go to run the app on my physical device.

## Challenges Faced
- Initially had a network issue because phone and laptop were on different Wi-Fi networks.
- Resolved by connecting both devices to the same network.
